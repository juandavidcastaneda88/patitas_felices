# 🐾 Patitas Felices — Sitio Web Juan David Castañeda
 
Sitio web de la **Fundación Patitas Felices**, una organización sin ánimo de lucro dedicada al rescate, rehabilitación y reubicación de animales en situación de vulnerabilidad. Fundada en 2015, la fundación ha rescatado más de 2.400 animales y conectado más de 1.800 familias adoptantes.

---
 
## 📁 Estructura del proyecto
 
```
patitas_felices/
* index.html          # Página principal del sitio
* perfil-animal.html  # Página de perfil individual de un animal en adopción
* css/
* style.css       # Estilos globales del sitio
* perfil.css      # Estilos específicos para la página de perfil
* img/                # Carpeta de imágenes locales
```
 
---
 
## 📄 Páginas
 
### `index.html` — Página principal
Contiene todas las secciones del sitio:
- **Hero / Banner** — Presentación de la fundación con carrusel de imágenes y estadísticas destacadas.
- **Resumen / Quick Actions** — Accesos rápidos a Adoptar, Donar, Voluntariado y Apadrinamiento.
- **Misión y Visión** — Descripción de los valores y objetivos de la fundación.
- **Galería** — Muestra fotográfica de los animales rescatados.
- **Adoptar** — Listado de animales disponibles para adopción.
- **Testimonios** — Historias de familias adoptantes.
- **Contacto** — Formulario de contacto e información de la fundación.
### `perfil-animal.html` — Perfil de adopción
Página de detalle de un animal específico (actualmente "Bongo", Pastor Alemán cruzado). Incluye galería de fotos, características del animal, estado de vacunación y formulario de solicitud de adopción.
